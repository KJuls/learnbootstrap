04 Working With Tables

In this lesson we'll learn about different tags are formatted in bootstrap and how different they look from the generic HTML rendering that we know

Defaults

  • The .table class adds basic styling (light padding and only horizontal dividers) to a table
  • The .table-responsive class creates a responsive table. The table will then scroll horizontally on small devices (under 768px). When viewing on anything larger than 768px wide, there is no difference

Contextual Classes

Element Description
.table-active Applies the hover color to the table row or table cell
.table-success Indicates a successful or positive action
.table-warning Indicates a warning that might need attention
.table-info Indicates a neutral informative change or action
.table-danger Indicates a dangerous or potentially negative action
