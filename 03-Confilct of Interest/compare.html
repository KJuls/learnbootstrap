Bootstrap Defaults vs. Browser Defaults

Alas! The <code> element!

The <small> element can be used to create a lighter, secondary text in headings

The <mark> element is used to highlight text.

The <abbr> element is used to mark up an abbreviation or acronym: The WHO was founded in 1948.

The <blockquote> element is used to present content from another source:

Ubi est secundus byssus? Cum fluctui mori, omnes humani generises manifestum peritus, audax nutrixes. Orexiss manducare! Est superbus capio, cesaris. Lotus, mirabilis tabess solite captis de salvus, superbus gabalium. cum fermium observare, omnes urbses perdere dexter, primus resistentiaes. unt extumes examinare barbatus, domesticus bullaes.

King James XII

To show the quote on the right, use the .blockquote-reverse class:
